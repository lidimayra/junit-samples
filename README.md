# JUnit samples

Samples to demonstrate how to work with [JUnit](https://junit.org/) (Java testing framework).

## Pre-requisites
- Java 8
- Maven 3

## Installing

Clone project:

```
git clone git@github.com:lidimayra/junit-samples.git
```

Install dependencies with [Maven](https://maven.apache.org/):
```
mvn clean install
```

Run test suite:
```
mvn test
```
